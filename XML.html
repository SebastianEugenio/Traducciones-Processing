Nombre

XML
Ejemplos
// el siguiente archivo XML llamado  "mammals.xml" es analizado 
// En el siguiente código. este debe estar en la carpeta "data" del proyecto.
//
// <?xml version="1.0"?>
// <mammals>
//   <animal id="0" species="Capra hircus">Goat</animal>
//   <animal id="1" species="Panthera pardus">Leopard</animal>
//   <animal id="2" species="Equus zebra">Zebra</animal>
// </mammals>

XML xml;

void setup() {
  xml = loadXML("mammals.xml");
  XML[] children = xml.getChildren("animal");

  for (int i = 0; i < children.length; i++) {
    int id = children[i].getInt("id");
    String coloring = children[i].getString("species");
    String name = children[i].getContent();
    println(id + ", " + coloring + ", " + name);
  }
}

// Sketch prints:
// 0, Capra hircus, Goat
// 1, Panthera pardus, Leopard
// 2, Equus zebra, Zebra
Descripcion XML Es una representación de un objeto XML, capaz de analizar código XML. Utilice loadXML () para cargar archivos XML externos y crear objetos XML .
Sólo los archivos codificados como UTF-8 (o ASCII normal) se analizan correctamente; El parámetro de codificación dentro de archivos XML se ignora.
Metodos
getParent() Obtiene una copia del elemento padre
getName() Obtiene el nombre completo de los elementos
setName() Establece el nombre del elemento
hasChildren() Comprueba si un elemento tiene o no hijos
listChildren() Devuelve los nombres de todos los hijos como una matriz
getChildren() Devuelve una matriz que contiene todos los elementos hijos
getChild() Devuelve el elemento hijo con el valor de índice especificado o ruta de acceso
addChild() Agrega un nuevo hijo al elemento
removeChild() Elimina el niño especificado
getAttributeCount() Cuenta el número de atributos del elemento especificado
listAttributes() Devuelve una lista de nombres de todos los atributos como una matriz
hasAttribute() Comprueba si un elemento tiene o no el atributo especificado
getString() Obtiene el contenido de un atributo como String
setString() Establece el contenido de un atributo como String
getInt() Obtiene el contenido de un atributo como un int
setInt() Establece el contenido de un atributo como un int
getFloat() Obtiene el contenido de un atributo como un flotante
setFloat() Establece el contenido de un atributo como un flotante
getContent() Obtiene el contenido de un elemento
getIntContent() Obtiene el contenido de un elemento como un int
getFloatContent() Obtiene el contenido de un elemento como un flotante
setContent() establece el contenido de un elemento
format() Formatea los datos XML como una cadena
toString() Obtiene datos XML como una cadena utilizando formato predeterminado
