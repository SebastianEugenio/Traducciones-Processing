Nombre

color
Ejemplos
color c1 = color(204, 153, 0);
color c2 = #FFCC00;
noStroke();
fill(c1);
rect(0, 0, 25, 100);
fill(c2);
rect(25, 0, 25, 100);
color c3 = get(10, 50);
fill(c3);
rect(50, 0, 50, 100);
Descripcion Tipo de datos para almacenar valores de color. Los colores se pueden asignar con get() y color() O pueden especificarse directamente usando la notación hexadecimal como #FFCC00 o 0xFFFFCCOO.

usando print() o println() en un color producirá resultados extraños (generalmente números negativos) debido a la forma en que los colores se almacenan en la memoria. Una mejor técnica es usar el hex(), función para formatear los datos de color, o usar red(), green(), y blue() funciones para obtener valores individuales e imprimir. El hue(), saturation(), y brightness() funcionan de manera similar. Para extraer los valores de rojo, verde y azul rápidamente (por ejemplo, al analizar una imagen o un marco de vídeo), utilicebit shifting.

Los valores también se pueden crear usando la notación de color web. Por ejemplo: color c = #006699

La notación de color Web sólo funciona para colores opacos. Para definir un color con un valor alfa, puede utilizar color () o notación hexadecimal. Para la notación hexadecimal, prefija los valores con 0x , por ejemplo color c = 0xCC006699 . En ese ejemplo, CC (el valor hexadecimal de 204) es el valor alfa, y el resto es idéntico a un color web. Tenga en cuenta que en la notación hexadecimal, el valor alfa aparece en la primera posición. (El valor alfa aparece por último cuando se utiliza en color () , relleno () y trazo () . De escribir el ejemplo anterior, pero usando color() y especificando el valor alfa como su segundo parámetro: color c = color (# 006699, 191)

Desde el punto de vista técnico, los colores son 32 bits de información ordenados como AAAAAAAARRRRRRRRGGGGGGGGBBBBBBBB donde los A contienen el valor alfa, los R son el valor rojo, los G son verdes y los B son azules. Cada componente es 8 bits (un número entre 0 y 255). Estos valores se pueden manipular con desplazamiento de bits.
